## 📄 Simple Portfolio Page


A clean, responsive portfolio landing page built with HTML and CSS. It showcases a personal profile, work samples, and FAQ-style sections using modern layout techniques.

---

## 🧱 Features

* Responsive layout for mobile, tablet, and desktop
* Hero section with profile introduction
* Image gallery grid layout
* FAQ section using native HTML details/summary
* Simple navigation with anchor links
* Mobile-first CSS design approach

---

## 🛠 Technologies Used

* HTML5
* CSS3
* Google Fonts
* Responsive CSS Grid & Flexbox

---

## 🚀 Installation

```bash
git clone https://github.com/krowey-richmond/my-portfolio.git
```

Navigate to:

```bash
projects/beginner/simple-portfolio-page
```

Open `index.html` in your browser.

---

## 💻 Usage

* Scroll through sections to view content
* Click navigation links to jump between sections
* Resize screen to test responsiveness
