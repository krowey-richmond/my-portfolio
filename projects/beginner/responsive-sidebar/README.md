# Responsive Sidebar Navigation

---

## Overview

This is a responsive sidebar navigation project built with HTML, CSS, and JavaScript. It demonstrates a collapsible sidebar, dropdown menus, and responsive layout behavior for both desktop and mobile screens.

## Features

- Collapsible sidebar navigation
- Dropdown submenu support
- Active link styling
- Responsive layout for mobile and desktop
- Sidebar toggle functionality
- Smooth UI transitions

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Usage

Open `index.html` in a browser to view the project.

---

## Responsive Behavior

- Desktop: Fixed sidebar navigation
- Mobile: Bottom navigation layout with icons only
