# Login & Signup Form System

## Overview
A clean authentication UI with both login and signup forms, including client-side validation for required fields, password strength, and matching passwords.

## Live Demo
https://your-live-link-here

## Features
- Login and signup pages
- Form validation (email, password, name checks)
- Password match verification
- Inline error messages
- Input highlight for invalid fields
- Responsive centered layout
- Simple UI with modern styling

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Setup
1. Clone the repository
2. Open `index.html` or `login.html` in a browser

## Notes
- Validation runs on the client side only
- No backend authentication connected yet
